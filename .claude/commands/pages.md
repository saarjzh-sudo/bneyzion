# /pages — כל דפי הפרויקט

פתח את קובץ `src/App.tsx` וחלץ את כל ה-Routes. הצג אותם כטבלה מסודרת עם קישורים לפרויקט בבדיקה המקומית (`http://localhost:8082`).

פורמט הפלט:

```
🗺️  דפי bneyzion — http://localhost:8082

ציבורי
  /                → http://localhost:8082/
  /parasha         → http://localhost:8082/parasha
  ...

ניסי המלחמה
  /dor-haplaot     → http://localhost:8082/dor-haplaot

אדמין
  /admin           → http://localhost:8082/admin
  ...

💡 ניווט ויזואלי: http://localhost:8082/dev-pages
```

לאחר הצגת הרשימה, שאל אם לפתוח דף מסוים בתצוגה מקדימה.
